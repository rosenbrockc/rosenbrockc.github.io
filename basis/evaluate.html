Basis Expansion Evaluation¶

Functions for evaluating basis functions for a given potential and constructing the Hamiltonian matrix for the basis expansion solution. By default, we use the Fourier \(\sin(n \pi x/L)\) basis functions as explained in [1].

[1]http://dx.doi.org/10.1119/1.4944706
basis.evaluate.H(V, N)[source]¶

Returns the Hamiltonian matrix for the specified potential so that it can be solved via basis expansion. Assumed K-P form of the potential.

Parameters:
  • V (basis.potential.Potential) – object for evaluating the potential.
  • N (int) – number of basis functions to use.
Returns:

with shape (N, N); and elements as specified in
