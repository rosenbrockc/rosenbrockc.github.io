Database Builder¶

Database builder is a script that gets installed to matdb_build.py in your user’s bin. It grants high-level access to certain sequences of API calls needed to construct the databases from the matdb.yml Database Specification File.

matdb.scripts.matdb_build.examples()[source]¶

Prints examples of using the script to the console using colored output.

matdb.scripts.matdb_build.run(args)[source]¶

Runs the matdb setup and cleanup to produce database files.

matdb.scripts.matdb_build.script_options = {‘–status’: {‘action’: ‘store_true’, ‘help’: ‘Determines status of the databases based on presence of completed VASP directories. Sanity check before `-x`.’}, ‘dbspec’: {‘help’: ‘File containing the database specifications.’}, ‘-c’: {‘action’: ‘store_true’, ‘help’: ‘Cleanup the databases that have completed execution so that results can be extracted.’}, ‘-x’: {‘action’: ‘store_true’, ‘help’: ‘Submit the job array file for each database that has folders ready to run.’}, ‘-s’: {‘action’: ‘store_true’, ‘help’: ‘Run the setup method for each database.’}}¶
